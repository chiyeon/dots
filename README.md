# dots
a collection of my personal dot files &amp; general linux configuration

### dependencies
- kitty (terminal)
- zsh
- micro (text editor)

### other software i normally get
- ranger (file explorer)
- cmus (music)
- neofetch / bunnyfetch / pfetch (system info fetchers)
- rofi (system menus)
- polybar (bar)
- feh / nitrogen (wallpapers)
- picom (compositor)

### more info
- zsh uses the Pure theme
- kitty uses the Later This Evening colors
